# РекламаСтрой Designer v2.0 Exact + Autosave

Добавлено:
- режим «Точный» после калибровки по 2 точкам;
- статус «Предварительная визуализация», если масштаб ещё не задан;
- отображение масштаба: см → пиксели, мм/px, примерная точность;
- автосохранение проекта в браузере через localStorage;
- сохранение текста, цветов, шрифта, фасада, логотипа, подсветки, ночного режима, высоты букв и точек калибровки.

Сборка:
- Framework preset: React (Vite)
- Build command: npm run build
- Build output directory: dist
