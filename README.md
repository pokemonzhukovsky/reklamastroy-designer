# Онлайн-конструктор вывесок РекламаСтрой

Онлайн-конструктор вывесок разработан компанией **РПК «РекламаСтрой»** для предварительной визуализации наружной рекламы на фотографии фасада здания. Инструмент помогает быстро примерить будущую вывеску, оценить примерные габариты, расположение текста, логотипа, цветовые решения и подготовить PDF-файл для дальнейшего расчета.

Конструктор предназначен для клиентов компании «РекламаСтрой», менеджеров, дизайнеров и специалистов, которые работают с проектами наружной рекламы: световыми вывесками, объемными буквами, логотипами, фасадными надписями и рекламными конструкциями.

## Для чего нужен конструктор

С помощью конструктора можно:

- загрузить фотографию фасада, входной группы, витрины или другой поверхности;
- указать известный размер на фото, например ширину окна, двери или витрины;
- откалибровать масштаб по двум точкам;
- добавить текст будущей вывески;
- включить вторую строку текста;
- выбрать шрифт, цвет лицевой части и цвет борта;
- загрузить логотип;
- разместить логотип слева, справа, сверху или снизу от текста;
- посмотреть визуализацию в дневном и ночном режиме;
- сформировать PDF-файл с визуализацией, размерами и основными параметрами вывески.

## Как пользоваться

1. **Загрузите фото фасада**  
   Используйте качественную фотографию фасада, сделанную максимально прямо, без сильных искажений перспективы. Чем ровнее фото, тем точнее будет предварительная визуализация.

2. **Укажите известный размер на фото**  
   Найдите на фотографии объект, реальный размер которого известен: окно, дверь, витрина, проем, фасадная панель или другой элемент. Поставьте две точки по краям этого объекта и укажите его реальный размер в миллиметрах.

3. **Введите текст вывески**  
   Укажите основную строку будущей вывески. При необходимости добавьте вторую строку и настройте для нее отдельный шрифт, цвет лицевой части, цвет борта и выравнивание.

4. **Настройте размер букв**  
   Укажите высоту букв в миллиметрах. Конструктор использует введенный масштаб и показывает вывеску на фотографии в заданном размере.

5. **Выберите цвета ORACAL**  
   Для лицевой части и бортов можно выбрать цвета из палитр ORACAL. Цветовые плашки в интерфейсе помогают быстрее ориентироваться в вариантах.

6. **Добавьте логотип**  
   Можно загрузить логотип в поддерживаемом формате и указать его размер. Логотип можно разместить слева, справа, сверху или снизу относительно текста.

7. **Проверьте визуализацию**  
   Оцените, как вывеска смотрится на фасаде. При необходимости измените размер, шрифт, цвет, положение или композицию.

8. **Скачайте PDF**  
   PDF-файл можно использовать как предварительный материал для обсуждения проекта, расчета стоимости и согласования параметров будущей вывески.

## Важное предупреждение

Визуализация в конструкторе является предварительной. Точность результата зависит от качества фотографии, правильности выбранных точек калибровки и корректности введенных размеров. Итоговые технические решения, конструктив, материалы, способы крепления, электрика, согласование и монтаж должны проверяться специалистами компании «РекламаСтрой».

PDF-файл, сформированный в конструкторе, не является рабочей конструкторской документацией и не заменяет профессиональный проект вывески.

## Права на использование

Онлайн-конструктор вывесок является разработкой компании **РПК «РекламаСтрой»**.

Запрещается без письменного согласия компании «РекламаСтрой»:

- копировать данный конструктор;
- размещать его на сторонних сайтах;
- вставлять его через iframe на другие ресурсы;
- использовать код, интерфейс или PDF-шаблоны в коммерческих целях;
- изменять название, логотип, водяные знаки и фирменные элементы;
- распространять архив, исходный код или собранную версию третьим лицам.

Использование конструктора разрешено только на официальных ресурсах компании «РекламаСтрой» и в рамках задач, связанных с изготовлением и расчетом наружной рекламы.

## Публикация на GitHub Pages

Проект является статическим React/Vite-приложением и может быть опубликован через GitHub Pages.

Основные параметры публикации:

- Build command: `npm run build`
- Build output directory: `dist`
- Рабочий workflow: `.github/workflows/deploy.yml`
- Для репозитория `reklamastroy-designer` в `vite.config.ts` должен быть указан `base: '/reklamastroy-designer/'`

После успешной публикации конструктор можно вставить на страницу сайта через HTML-блок/iframe.
