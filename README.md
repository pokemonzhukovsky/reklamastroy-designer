# ReklamaStroy Designer v3.18 — empty size inputs

Изменения в этой версии:
- в полях размеров теперь можно полностью удалить значение;
- при очистке поля больше не подставляется автоматическая единица;
- новое значение применяется сразу после ввода корректного числа больше нуля;
- сохранены предыдущие правки: точный размер логотипа, исправленные стрелки, размещение логотипа слева/справа/сверху/снизу.

Cloudflare Pages:
- Build command: `npm run build`
- Build output directory: `dist`


Обновление:
- добавлена маленькая кнопка «Добавить новое фото» сверху справа на рабочем экране визуализации;
- кнопка открывает загрузку нового фото фасада и не попадает в PDF.
