# ReklamaStroy Designer 3.1 Modern UI

Чистая версия без `node_modules`, `dist`, `package-lock.json`.

## Что внутри
- React + Vite
- современный интерфейс в стиле профессионального SaaS-конструктора
- левая вертикальная панель этапов
- верхняя инфографика процесса
- карточки ORACAL и подсветки
- калибровка по 2 точкам
- режим «Точный»
- автосохранение
- PDF

## Cloudflare Pages
- Build command: `npm run build`
- Output directory: `dist`
