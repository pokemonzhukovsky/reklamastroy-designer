# РекламаСтрой Designer — Foundation v1.0

Профессиональная основа React/Vite-приложения для онлайн-конструктора объёмных букв.

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка для Cloudflare Pages

```bash
npm run build
```

Cloudflare Pages:
- Framework preset: Vite
- Build command: `npm run build`
- Build output directory: `dist`

## Назначение

Проект является основой для конструктора на странице `reklamastroy.ru/constructor`.
